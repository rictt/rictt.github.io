## 基本使用

### 链接/路由跳转
  假设你现在在```foo/one.md```中：

  ```md
  [Home](/) <!-- 跳转到根部的README.md -->
  [foo](/foo/) <!--跳转到foo文件夹下的README.md -->
  [foo heading](./#heading) <!-- 跳转到foo下的heading-->
  ```

### 表格

  ```md
  | Tables        | Are           | Cool  |
  | ------------- |:-------------:| -----:|
  | col 3 is      | right-aligned | $1600 |
  | col 2 is      | centered      |   $12 |
  | zebra stripes | are neat      |    $1 |
  ```
